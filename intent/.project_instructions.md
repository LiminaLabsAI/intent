# Cerebrio Intent Lifecycle App

## Project Overview
Full-featured Intent Lifecycle Management system implementing the 7-stage Cerebrio workflow: Intent Capture → Parsing → Semantic Understanding → Normalization → Quality Gate → Approval Decision Engine → Intent ID Creation.

## Tech Stack
- App Router, TypeScript, Tailwind CSS
- PostgreSQL with Prisma ORM
- NextAuth.js v4 (credentials provider)
- LLM API (gpt-5.4-mini) for stages 2-6 processing via streaming SSE

## Visual Style
- Light/white theme: Clean white backgrounds with gray-50 sidebar
- Primary accent: Blue (`#3b82f6`) + Indigo (`#6366f1`) gradients
- Minimal color usage — mostly grayscale with blue accents
- ChatGPT-style layout: collapsible sidebar + main content area
- Fonts: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (code/IDs)
- Product name: **Flow** (renamed from Cerebrio)
- Logo: Custom SVG wave/flow icon with blue-to-indigo gradient

## User Roles
- **ADMIN**: Full access - user management, stats, all intents, reviews
- **REVIEWER**: Review queue access, can decide on intents
- **END_USER**: Submit intents, view own intents only

## Key Pages
- `/dashboard` - ChatGPT-like interface for submitting intents
- `/intent/[id]` - Full lifecycle detail view with stages, comments, audit
- `/registry` - Searchable/filterable intent registry with CSV export
- `/reviews` - Review queue for ADMIN/REVIEWER roles
- `/admin` - Admin dashboard with stats and user management

## Architecture
- Streaming SSE pipeline: `/api/intents/[id]/process` runs stages 2-6 sequentially via LLM
- Each stage updates the Intent record in DB and streams progress events
- Decision outcomes: AUTO_APPROVED, NEEDS_CLARIFICATION, HUMAN_REVIEW_REQUIRED, CONDITIONAL_APPROVAL, REJECTED
- Auto-generates Intent ID (INT-XXXXX) for approved intents
- Version history + audit trail on all changes
